# doc
Loft p2p messenger protocol

## Description
Открытый протокол обмена сообщениями
Основные участники:
- клиенты (мобильные и настольные устройства, тд)
- ноды (серверы учавствующие в пересылке сообщений)

## Типы информационных каналов
### users
users - информация о пользователях - генерируется и обновляется клиентским (пользовательским устройством), реплицируется нодами

### sessions
session - обмен сообщениями в рамках сессий
сессии могут быть как на 1-2 пользователей, так и на множество

### blob
хранение медиаданных
